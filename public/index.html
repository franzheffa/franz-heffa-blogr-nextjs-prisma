Buttertech – passerelle d’agents IA pour l’entreprise

Plateforme d’agents IA spécialisés (recherche, RAG, automatisation) orchestrés derrière un API Gateway (Cloud Run), avec un front léger (Vercel). Objectif : fiabilité, observabilité, isolation par agent.

Voir la démo /api/health /gateway/health

Ce que nous faisons

  • Gateway FastAPI qui proxifie des agents isolés : /agents/<nom>
  • Agents unitaires (ex. echo, gemini) packagés & déployés indépendamment
  • Front minimal (Vercel Functions + static) pour tests & intégrations rapides
  • Observabilité (logs Cloud Run & Vercel), sécurité par service account & env par agent

Problèmes résolus

  • Éviter les conflits entre agents via isolation
  • Réduire le time-to-prod : build & deploy en 1 commande (Cloud Build)
  • Offrir une surface API stable pour intégrations web & internes

Produits (état actuel)

  • Agent Gateway (Cloud Run) — routes : /agents/echo, /agents/gemini
  • Agent “Gemini” — génération texte + image (Vertex AI, gemini-2.x)
  • Voix — synthèse via Cloud Text-to-Speech
